** Student Tracker Management System

Student Tracker is a comprehensive student management system that tracks students' academic performance and attendance.
It allows you to add, remove, and search for students, calculate the class GPA average, generate an attendance report, 
and list top-performing students (honor roll).

** Features
- Add or remove students.
- Search for a student by name or ID.
- Calculate and display class GPA average.
- Generate attendance reports (percentage attendance per student).
- List top-performing students.
- Exit the program.

** Usage 
When using the program, you are prompted to choose one of eight options:

1.*Adding a new student*
In this option you will add a new student by entering their student ID, name, GPA, days present in school, and the total days 
of school.

2.*Removing a student*
In this option you will remove a student by entering their student ID, this will remove a student and free up a spot for a new student to be added.

3.*Display all students*
In this option you will display a list of all current students. The list will in order print out students ID, name, GPA, 
and attendance in percentage that is calculated by the number of days a student is present divided by the total number of days in the school year times 100. However, if there are no existing students in the student tracker system, you will receive the following message, "No available students to display."

4.*Search for a student* This option is similar to the third but more precise. In this option you will search for an individual student by either their
name or ID and will print out that specific student's information. If, however, the name or ID you are attempting to search for does not exist in the 
student tracker system, you will be displayed the message, "No matching student found."

5.*Calculating the average GPA in a class*
In this option, all the GPA's of students in a class will be added together and divided by the amount of 
students in that class to get the average GPA of the classroom. Note: This will not function if there are no existing students in the class.

6.*Generating attendance report*
In this option, a list of attendance for each student will be displayed. This option will calculate each student's attendance by dividing the total number of days a student is present in school by the total of days there are in a school year times 100. Please note: if there are no existing students in the student tracker system, you will be displayed with a message, "There are no students available for an attendance report."

7.*Listing all Honor Roll Students*
In this option, all students with a GPA greater than or equal to 3.5 will be displayed. Note: If there are no existing students in the student tracker system, the message, "There are no current students to list an honor roll," will be displayed.

8.*Leave the student tracker system program*
After using the student tracker system, this option will exit you out of the program.
